

# Stackblitz

## Setup 2 = online = Stackblitz online IDE and container

Stackblitz provides a full onlne IDE (like VSCode) and a web application container

- so you can see changed to your web application as soon as you save a chance to a file.


```{image} /02_quickstart/images/stack_blitz.png
:alt: Screenshot showing interactive online Svelte project in Stackblitz.com webpage
```
<!-- :width: 50% -->


To get started do the following:

1. Visit [stackblitz.com], **Sign In** with Google or Github

- (or create an account using your email address)

2. Click the **Dashboard** top menu option and create a new Svelte (Javascript) project


```{image} /02_quickstart/images/stack_blitz_new_svelte.png
:alt: Screenshot showing new Svelte (Javascript) project in Stackblitz.com
```
<!-- :width: 50% -->


3. stack hacking some Svelte :-)
