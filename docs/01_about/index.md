# Introduction

Welcome to the learning materials for the **Front-End Dev** module.

## About the module

Some key goals of the module:

- **Recap web dev skills**

    - a recap of HTML, CSS and JavaScript
  
- **Recap of 'how the web works**

    - Front End implies there is a 'Back End' - recap the basics of modern web application architectures

- **Svelte and alternatives**

    - You'll learn to design and develop web page user-interfaces (UIs) wit the Svelte library/framework. Learn why this has been chosen from a range of alternatives...

- **UI components, events and event handlers**

    - while learning Svelte, you'll core UI programming concepts and patterns

        - interactive web elements, events (mouse over, clicking, typing a character)

        - event handler functions (code to do SOMETHING when an event is recieved by a compoent), 

        - variable 'state', and linking state changed to immediately UI elememnt change

- **UX**

    - concepts about how to design and implment for high quality User Experience (UX)

*Have fun...*
