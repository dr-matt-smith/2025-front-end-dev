# Introduction

Welcome to the learning materials for the Front-End Dev module.

## About the module

A multi-purpose workbench application is a complex tool and difficult to present in a few words. However, here are some of the key features of Celbridge:

- **Recap web dev skills**
  : - a recap of HTML, CSS and JavaScript
  
- **Recap of 'how the web works/**
  : - Front End implies there is a 'Back End' - recap the basics of modern web application architectures

- **Svelte and alternatives**
  : - You'll learn to design and develop web page user-interfaces (UIs) wit the Svelte library/framework. Learn why this has been chosen from a range of alternatives...


*Have fun...*
