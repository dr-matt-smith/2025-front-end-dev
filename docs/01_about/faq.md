## What code editor should I use?

It's up to you :-)

But, personally, for JS/web dev I use either [Jetbrains Webstorm]( https://www.jetbrains.com/webstorm/), or [Visual Studio Code](https://visualstudio.microsoft.com/downloads/)


:::{tip}
If you sign up for [Github Student Developer Pack](https://education.github.com/pack) then you can access the full professional versions of all the Jetbrains IDEs, including [Jetbrains Webstorm]( https://www.jetbrains.com/webstorm/)
:::



## Can I work on Linux / MacOS, or do I have to use Windows?

The great thing about web dev is that you can author your project on whatever system you prefer (Windows, MacOS, Linux, online IDE). Since web dev project are a publishing workflow (author / build for the web / publish & run on a web server)
